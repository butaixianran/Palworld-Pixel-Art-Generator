# Palworld Pixel Art Generator (幻獣パル Pixel Art 生成器)
パルワールド（Palworld）のセーブデータを編集し、ゲーム内のプレイヤー位置を起点として、巨大なピクセルアートの看板を作成します。お好みの画像をゲーム内に表示させることが可能です。

# インストール
* **Steam版のみ対応**
* **Windows**: リリースページから最新版をダウンロードし、解凍して実行してください。
* **その他のプラットフォーム**: 「ソースコードからの実行」セクションを参照してください。

# 事前準備
## テンプレートの準備
ゲーム内で、使用したいスタイルの「土台」「支柱」「壁」が少なくとも各1つずつ存在することを確認してください。

例えば、デフォルト設定では「未来スタイルの土台」を使用し、その上に「ガラスの支柱」を設置して写真壁の高さを上げ、さらに「未来スタイルの壁」を写真壁のキャンバスとして使用し、色を塗ります。

プログラムはセーブデータ内を検索し、未来スタイルの土台、ガラスの支柱、未来スタイルの壁が既に存在するかを確認します。それらのデータをコピーしてテンプレートとして使用し、新しい構造物を作成します（ゼロからデータを構築するわけではありません）。

![](screenshot/prepare.jpg)

これらのデータ構造は非常に複雑で、すべてのフィールドをリバースエンジニアリングして解析するのは時間と労力がかかり、ゲームのアップデートによって構造が変わる可能性もあります。そのため、ゲーム内に存在するデータをテンプレートとしてコピーする方法が最も簡便で安全です。

したがって、ゲームの世界にテンプレートとなる各スタイルの土台、支柱、壁が**少なくとも1つ存在している必要があります**。


# 使い方
## セーブデータの場所
```
%localappdata%\Pal\Saved\SaveGames\YOURID\RANDOMID\
```

## Windows
### ドット絵を生成
* Level.sav が格納されているディレクトリに、Players サブディレクトリが存在することを確認してください。その中には、座標（位置情報）を含むキャラクター情報が保存されています。
* プログラムを実行し、画面の指示に従って `Level.sav` ファイルを選択します。
* キャラクターリストの読み込みを待ち、キャラクターを選択してください。
* 画像を選択します。
* 一番下の実行ボタンをクリックし、完了を待ちます。処理には時間がかかるため、そのままお待ちください。
* アプリが元のセーブファイルを自動的にバックアップします。
* ゲームを起動して確認してください。

### ドット絵の壁を消去

アプリの画面で消去したい半径の範囲を入力してください。削除ボタンをクリックするとセーブデータが書き換えられ、キャラクターを中心に、指定した半径内にある特定のスタイルの土台、支柱、壁が削除されます。

または、建造物が時間の経過とともに自然に風化するのを待ちます。

## ソースコードからの実行
### ソースコードのダウンロード
* gitがインストールされている場合：
  ```
  git clone https://github.com/butaixianran/Palworld-Pixel-Art-Generator
  ```
* または、GitHubの緑色の「Code」ボタンをクリックし、「Download ZIP」を選択してください。

### 環境構築
* Pythonをインストールします（専用の仮想環境の作成を推奨）。
* C++ビルド環境をインストールします。Windowsの場合は、Visual Studio 2022で「C++によるデスクトップ開発」ワークロードを選択してください。

### セットアップ
* コマンドラインでプロジェクトの `src` ディレクトリに移動します。
* 以下のコマンドを実行します：`pip install -r requirements.txt`

### 実行
* `src` ディレクトリで以下のコマンドを実行します：`python main.py`

# パラメータ仕様
アプリのインターフェース上で各パラメータの説明を確認できますが、ここでは補足事項のみ記載します。

* **最大幅・最大高さ**: 画像のアスペクト比は自動的に維持され、画像が引き伸ばされることはありません。
* **透明ピクセル**: 透明なピクセルを表現するために、黒く塗られた「ガラスの壁」を使用します。黒いガラスは透明度が最も高く、透明ピクセルの表現に適しています。
